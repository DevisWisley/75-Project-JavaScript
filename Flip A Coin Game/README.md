# Flip A Coin Game
Flip A Coin Game serves as a tool for a random decider or simple decision maker between two options.

## Flip A Coin Game typically allow you to:
* Avoiding occurrences or disagreements when having to choose between two things.
* Adding an element of luck or unpredictability to other games.
* Helps make small decisions that don't require complicated thinking.
